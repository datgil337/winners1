CSS란 무엇인가?

The CSS Transforms Module defines 2D transforms (rotations, translations, etc.)
that can be applied to elements after the normal layout
(i.e., a transform does not affect the position of surrounding elements).
The transforms are also available in SVG (as values for the transform attribute)
and the specification is a joint work of the CSS and SVG working groups
